# Embedded-Main

> 西安邮电大学硬件科技协会

面向硬协嵌入式小组的主仓库，包含面经八股、项目资料、学习笔记、课程资料等。

---

## 快速开始

### 方式一：只想看看（推荐新人）

直接点进上面的文件夹浏览，或按 `T` 键全局搜索文件名。

### 方式二：想提交资料

1. **Fork 本仓库**（点右上角 Fork 按钮）
2. **克隆你 Fork 的仓库**：
   ```bash
   git clone https://github.com/你的用户名/Embedded-Main.git
   ```
3. 按 [CONTRIBUTING.md](./CONTRIBUTING.md) 的流程提交

> 不需要管理员授权。Fork 后就是你的仓库，改完提 PR 即可。

## 仓库内容

| 分类 | 路径 | 说明 |
|------|------|------|

## 分支说明

| 分支 | 用途 | 谁操作 |
|------|------|--------|
| `main` | 精选资料、通用内容，面向所有人 | 维护人合并 |
| `2024` | 2024 届成员提交的资料 | 维护人合并 |
| `2025` | 2025 届成员提交的资料 | 维护人合并 |
| `2026` | 2026 届成员提交的资料 | 维护人合并 |
| `feature/xxx` | 你 Fork 后自己创建的临时分支 | 你自己 |

**规则**：
- 质量评级高的资料，维护人合并到 `main`
- 一般资料合并到对应届的分支
- 不需要自己开长期分支，用完即删

## 提交方式（二选一）

### 推荐：GitHub 网页直接编辑

适合只改几个字、传个文件的情况：
1. 打开仓库里的文件
2. 点右上角铅笔图标 ✏️ 编辑
3. 底部填写修改说明，直接提交 PR

### 完整流程：Fork + Pull Request

适合新增内容、批量修改，详见 [CONTRIBUTING.md](./CONTRIBUTING.md)。

## 文档格式

Word/PDF 只用于老师下发的课件等无法转录的文件。
